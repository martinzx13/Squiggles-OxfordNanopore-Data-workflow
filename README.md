# Squiggles-OxfordNanopore-Data-workflow

This is a repository that wants to make a data workflow 
for Squiggles downloading from a dataset, making an 
  alignment with dorado and creating a bam and sam file 
  to be able do other analisys.

File structure.

src - data - raw  # Data with the Squeggles. 
           - processed #Data with the sam and bam files
    - code - download.sh # Script that will create the folders install dependencies and download the data.
           - processes # Script that will run the analisys and prepare the files. 
 
- docs / some articles, for understand the use of the tool.
